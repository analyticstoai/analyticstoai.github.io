EzzyTalk.

The Problem
EzzyShip company is sending its customer’s order status on their personal email id, but the customer missed to check the mails or somehow missed, so customer repetitively calling and sending a message to EzzyShip company for his/her odder status, that cost extra to the company for hiring new customer executive’s team & that also not support 24*7.

The Solution.

A2ai created EzzyShip Bot, which has Whats- App integration and sends real-time customer status on his/her mobile number. This is a two-way communication Chat-Bot which answer 90 % of the customer queries on a real-time basis. This tool has all the functionality like



  • Real-time report generation
  • Individual customer’s order status
  • To add new customer details, time log
  • Integrated with the Facebook API to answer real time queries from Facebook

    • Impact.



  • Huge cost saving to the company
  • 24x7 support
  • Real-time report generation
  • Real-time order’s status to customer
  • Enhance customer satisfaction
